# Laboratorio: Tecnologías JavaScript y AJAX

Laboratorio práctico enfocado en el procesamiento de lógica en el cliente y comunicación asíncrona mediante AJAX.

## Descripción del Proyecto
Este proyecto es un tablero único que centraliza cuatro ejercicios de lógica algorítmica y un módulo de inspección de red. 

- **Lógica:** Implementación de funciones para detectar palíndromos, comparar números, extraer vocales únicas y contar frecuencias de caracteres.
- **AJAX:** Módulo para realizar peticiones HTTP asíncronas, permitiendo visualizar el flujo de estados de la conexión, cabeceras y contenido recibido del servidor.

## Tecnologías Utilizadas
- **HTML5:** Estructura semántica.
- **CSS3:** Diseño responsivo (Flexbox/Grid).
- **JavaScript (Vanilla):** Lógica del negocio y manipulación del DOM.
- **AJAX (XMLHttpRequest):** Comunicación asíncrona para inspección de peticiones.
