# Laboratorio de JavaScript: Ejercicios Algorítmicos

## 📋 Descripción General
Este proyecto es una práctica de desarrollo web básico que recopila la resolución de 4 ejercicios lógicos y de manipulación de cadenas utilizando **JavaScript nativo**. Toda la estructura, el diseño visual y la lógica están unificados en un único archivo para garantizar su correcto funcionamiento y compatibilidad en entornos de despliegue directo.

---

## 🎯 Objetivos del Laboratorio
* Aplicar condicionales y estructuras de control básicas (`if`, `else`).
* Implementar bucles e iteraciones tradicionales (`for`).
* Manipular elementos del DOM para capturar datos de entrada y mostrar resultados en pantalla.
* Normalizar y validar cadenas de texto de forma manual.
